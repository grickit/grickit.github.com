GambCiv
Downloads

GambCiv is still in a pre-release state. So most of these download links are inactive.

Once it is officially released you will, of course, be able to find it via the addon interface inside the game.

To install these, download the archive and extract them into the the add-ons part of your userdata folder.

Latest stable version

  • Wesnoth.org (0mb) (.tar.bz2)
  • Github.com (0mb) (.zip)
  • Github.com (0mb) (.tar.gz)

Bleeding edge version*

  • Github.com (1mb) (.zip)
  • Github.com (1mb) (.tar.gz)
*The bleeding edge version contains all of the new changes. It may have numerous bugs. It also changes frequently and will not be compatible with other versions for online play.

Clone with git

You can get GambCiv via git as well. Just link the repo right into your userdata folder.

Linux instructions:

$ git clone git://github.com/grickit/GambCiv.git
$ ln -s GambCiv/ ~/.local/share/wesnoth/1.9/data/add-ons/GambCiv
