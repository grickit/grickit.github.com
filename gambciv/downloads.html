GambCiv
Downloads

You can install GambCiv via the addon interface inside the game.

Or you can use one of these download links. Download the archive, and extract it into your userdata folder.

Latest stable version - 0.3 (beta)

  • Wesnoth.org (<1mb) (.tar.bz2)
  • Github.com (<1mb) (.zip)

Bleeding edge version*

  • Github.com (<1mb) (.zip)
  • Github.com (<1mb) (.tar.gz)
*The bleeding edge version contains all of the new changes. It may have numerous bugs. It also changes frequently and will not be compatible with other versions for online play.

Clone with git

You can get GambCiv via git as well. Just link the repo right into your userdata folder. All the restrictions of the bleeding edge version still apply.

Linux instructions:

$ git clone git://github.com/grickit/GambCiv.git
$ ln -s GambCiv/ ~/.local/share/wesnoth/1.9/data/add-ons/GambCiv
